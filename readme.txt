>マップ班へ
map.cは基本的にコピペだけど、追加した機能である摩擦とベルトコンベアについて説明するよ
(1)摩擦
	45行目のdouble friction[][5]に設定するよ
	例えば{ 2.0, 3.0, 15.0, 25.0, 0.05 }のときは
	2.0<プレイヤーのx座標<3.0 かつ 15.0<プレイヤーのy座標<25.0　のとき　プレイヤーが動く速度を0.05にする
	という意味だよ
	ここで、friction[][]の0行目はデフォルト値なので変えないでください
	また、プログラムの都合上、速度を変える範囲として指定したところが複数行で被ったときは、上の行の記述が優先されるよ
	例   { 2.0, 3.0, 15.0, 25.0, 0.05 },
	     { 0.0, 5.0, 20.0, 21.0, 0.02 }  の順で指定したとき
	     二つの範囲が被ってる　　2.0<x<3.0 かつ 20.0<y<21.0  では
	     上の行が優先されるので速度0.05になるよ
	fricindex(friction[][]の行数)も忘れないでね
(2)ベルトコンベア
	52行目のdouble belt[][6]に設定するよ
	0~3列目は範囲指定で、やり方は摩擦の時と同じだよ
	4列目はベルトコンベアの速さ、5列目はベルトコンベアの向き(1ならx軸正方向、2ならy軸正方向)を表すよ
	例えば4列目が(-0.05)、5列目が1.0のときは x軸負方向(左向き)に速さ0.05で動くベルトコンベアになるよ
下の方のコメントアウトを外せば、map.c単体で静止画が描けるよ
